# مثال‌های آسان برای مبتدیان در هوش مصنوعی

## 🚀 شروع کار

### پیش‌نیازها

اطمینان حاصل کنید که پایتون نصب شده است (نسخه ۳.۸ یا بالاتر توصیه می‌شود). بسته‌های مورد نیاز را نصب کنید:

```bash
# برای اسکریپت‌های پایتون
pip install numpy

# برای نوت‌بوک‌های Jupyter (طبقه‌بندی‌کننده تصویر)
pip install jupyter numpy pillow tensorflow
```

یا از محیط conda موجود در برنامه اصلی استفاده کنید:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### اجرای مثال‌ها

**برای فایل‌های پایتون (.py):**
```bash
python 01-hello-ai-world.py
```

**برای دفترچه‌های Jupyter (.ipynb):**
```bash
jupyter notebook 03-image-classifier.ipynb
```
