# စတင်သုံးရန်အဆင်ပြေသော AI ကိုအသုံးပြုရန် နမူနာများ

ကြိုဆိုပါသည်! ဤဖိုလ်ဒါတွင် AI နှင့် စက်ရုပ်သင်ယူမှုကို စတင်လေ့လာနိုင်ရန် ရိုးရှင်း၍ သပ်သပ်ရပ်ရပ် နမူနာများ ပါဝင်သည်။ နမူနာတိုင်းကို စတင်သုံးသူများအတွက် အသင့်ပြင်ထားပြီး အသေးစိတ် မှတ်ချက်များနှင့် ခြေလှမ်းလိုက်ရှင်းပြချက်များပါဝင်ပါသည်။

## 🚀 စတင်အသုံးပြုခြင်း

### မတိုင်မီ လိုအပ်ချက်များ

Python ကို တပ်ဆင်ထားသည်ကို သေချာစေပါ (3.8 နှင့်အထက်အကြံပြုသည်)။ လိုအပ်သော package များကို တပ်ဆင်ပါ။

```bash
# Python စက္ရုပ်များအတွက်
pip install numpy

# Jupyter စာအုပ်များအတွက် (ပုံများ စစ်ဆေးသူ)
pip install jupyter numpy pillow tensorflow
```

ဒါမှမဟုတ် မြောက်မြားသော သင်တန်းအတွင်းပါဝင်သော conda ပတ်ဝန်းကျင်ကို အသုံးပြုနိုင်သည်။

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### နမူနာများကို လုပ်ဆောင်ခြင်း

**Python စကရစ်(.py ဖိုင်) များအတွက်:**
```bash
python 01-hello-ai-world.py
```

**Jupyter မှတ်စုစာအုပ်(.ipynb ဖိုင်) များအတွက်:**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 သင်ယူရေးလမ်းကြောင်း

နမူနာများကို အဆင့်လိုက် နားလည်ကြည့်ပါ။

1. **"Hello AI World" နှင့် စတင်ပါ** - ပုံစံသိရှိခြင်း အခြေခံကို သင်ယူပါ
2. **ရိုးရှင်းသော Neural Network တည်ဆောက်ပါ** - Neural Network များသည် မည်သို့ လုပ်ဆောင်ကြသည်ကို နားလည်ပါ
3. **ဓာတ်ပုံခွဲခြားစက်ကို ကြိုးစားပါ** - အမှန်တကယ် AI ကို တင်ပြခြင်း
4. **စာသားခံစားချက် ကို ခွဲခြားပါ** - သဘာဝဘာသာစကားဖြင့် လေ့လာမှုကို စူးစမ်းပါ
