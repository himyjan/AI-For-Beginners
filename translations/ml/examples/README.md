# തുടക്കക്കാരോട് സൗഹൃദം പുലർത്തുന്ന AI ഉദാഹരണങ്ങൾ

സ്വാഗതം! ഈ ഡയറക്ടറിയിൽ AIയും മെഷീൻ ലേണിങ്ങും ആരംഭിക്കാൻ സഹായിക്കുന്ന ലളിതവും സ്വതന്ത്രവുമായ ഉദാഹരണങ്ങൾ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്. ഓരോ ഉദാഹരണവും വിശദമായി കമന്റുകളും ഘട്ടം ഘട്ടമായ വ്യാഖ്യാനങ്ങളോടെയും തുടക്കക്കാരെ സഹായിക്കുന്ന വിധത്തിലാണ് രൂപകൽപ്പന ചെയ്തിരിക്കുന്നത്.

## 🚀 തുടക്കം കുറിക്കുക

### ആവശ്യമായ മുൻപരിചയം

നിങ്ങളുടെ കമ്പ്യൂട്ടറിലേക്ക് പൈതൺ ഇൻസ്റ്റാൾ ചെയ്തുള്ള വിവരം ഉറപ്പാക്കുക (3.8 അല്ലെങ്കിൽ അതിനുമുമ്പ് ശുപാർശ ചെയ്യപ്പെടുന്നു). ആവശ്യമായ പാക്കേജുകൾ ഇൻസ്റ്റാൾ ചെയ്യുക:

```bash
# പൈഥൺ സ്ക്രിപ്റ്റുകൾക്കായി
pip install numpy

# ജൂപ്പിറ്റർ നോട്ട്‌ബുക്കുകൾക്കായി (ഇമേജ് ക്ലാസിഫയർ)
pip install jupyter numpy pillow tensorflow
```

അല്ലെങ്കിൽ പ്രധാന കോഴ്‌സിൽ നിന്നും കോണ്ടാ എൻവയോൺമെന്റ് ഉപയോഗിക്കുക:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### ഉദാഹരണങ്ങൾ പ്രവർത്തിപ്പിക്കൽ

**പൈതൺ സ്‌ക്രിപ്റ്റുകൾക്കായി (.py ഫയലുകൾ):**
```bash
python 01-hello-ai-world.py
```

**ജൂപ്പിറ്റർ നോട്ട്ബുക്കുകൾക്കായി (.ipynb ഫയലുകൾ):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 പഠന മാർഗ്ഗരേഖ

നിർദേശിച്ചിരിക്കുന്ന ക്രമത്തിലും മാർഗ്ഗരേഖയിൽ ഉൾപ്പെടുന്ന ഉദാഹരണങ്ങൾ അനുഗമിക്കുക:

1. **"Hello AI World"-ൽ തുടങ്ങുക** - പാറ്റേൺ തിരിച്ചറിവിന്റെ അടിസ്ഥാനങ്ങൾ പഠിക്കുക
2. **ലളിതമായ തലച്ചോറ് ശൃംഖല നിർമ്മിക്കുക** - Neural Network എങ്ങനെ പ്രവർത്തിക്കുന്നുവെന്ന് മനസിലാക്കുക
3. **ചിത്ര ജനിതീകരണം ശ്രമിക്കുക** - യാഥാർത്ഥ്യ ചിത്രങ്ങളോടെ AI കാണുക
4. **വാചകത്തിന്റെ സംഭവം വിശകലനം ചെയ്യുക** - സ്വാഭാവിക ഭാഷാ പ്രോസസിംഗിനെ ആഴത്തിൽ പഠിക്കുക
