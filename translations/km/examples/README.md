# ឧទាហរណ៍ AI សម្រួលសម្រាប់អ្នកចាប់ផ្តើម

## 🚀 ចាប់ផ្តើម

### តម្រូវការមុនខ្លួន

សូមប្រាកដថាអ្នកបានដំឡើង Python (3.8 ឬខ្ពស់ជាងនេះបានណែនាំ)។ ដំឡើងបណ្ណាល័យដែលត្រូវការ៖

```bash
# សម្រាប់ស្គ្រីប Python
pip install numpy

# សម្រាប់សៀវភៅកំណត់ត្រា Jupyter (កម្មវិធីចាត់ថ្នាក់រូបភាព)
pip install jupyter numpy pillow tensorflow
```

ឬប្រើបរិយាកាស conda ពីមេរៀនស្នូល៖

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### រត់ឧទាហរណ៍

**សម្រាប់ស្គ្រីប Python (.py files):**
```bash
python 01-hello-ai-world.py
```

**សម្រាប់ក្រដាស់ Jupyter (.ipynb files):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 ផ្លូវកំណត់ការរៀន

យើងផ្តល់អនុសាសន៍ឲ្យអនុវត្តឧទាហរណ៍តាមលំដាប់ៈ

1. **ចាប់ផ្តើមជាមួយ "Hello AI World"** - រៀនមូលដ្ឋាននៃការទទួលស្គាល់លំនាំ
2. **សាងសង់បណ្តាញប៉ុស្តិ៍ប្រសាទសាមញ្ញ** - យល់ដឹងពីរបៀបដែលបណ្តាញប្រសាទដំណើរការ
3. **សាកល្បងអ្នកចាត់ថ្នាក់រូបភាព** - មើល AI ដំណើរការជាមួយរូបភាពពិត
4. **វិភាគអារម្មណ៍អត្ថបទ** - ស្វែងយល់អំពីដំណើរការភាសាស្វ័យប្រវត្តិ

- **កុំបារម្ភពីការយល់ដឹងទាំងមូល** - ការរៀនត្រូវការពេលវេលា
