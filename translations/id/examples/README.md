## 🚀 Memulai

### Prasyarat

Pastikan Anda telah menginstal Python (3.8 atau lebih tinggi disarankan). Instal paket yang diperlukan:

```bash
# Untuk skrip Python
pip install numpy

# Untuk notebook Jupyter (pengklasifikasi gambar)
pip install jupyter numpy pillow tensorflow
```

Atau gunakan lingkungan conda dari kurikulum utama:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### Menjalankan Contoh

**Untuk skrip Python (.py):**
```bash
python 01-hello-ai-world.py
```

**Untuk notebook Jupyter (.ipynb):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 Jalur Pembelajaran

Kami menyarankan mengikuti contoh secara urut:

1. **Mulai dengan "Hello AI World"** - Pelajari dasar pengenalan pola
2. **Bangun Jaringan Saraf Sederhana** - Pahami cara kerja jaringan saraf
3. **Coba Pengklasifikasi Gambar** - Lihat AI beraksi dengan gambar nyata
4. **Analisis Sentimen Teks** - Jelajahi pemrosesan bahasa alami
