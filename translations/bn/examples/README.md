# শিক্ষানবীশ-অনুকূল AI উদাহরণসমূহ

স্বাগতম! এই ডিরেক্টরিটি সহজ, স্বতন্ত্র উদাহরণ ধারণ করে যা আপনাকে AI এবং মেশিন লার্নিং শুরু করতে সহায়তা করবে। প্রতিটি উদাহরণ শিক্ষানবীশ-অনুকূলভাবে বিস্তারিত মন্তব্য এবং ধাপে ধাপে ব্যাখ্যার সাথে ডিজাইন করা হয়েছে।

| উদাহরণ | বিবরণ | কঠিনতা | পূর্বপ্রয়োজনীয়তা |

## 🚀 শুরু করা

### পূর্বপ্রয়োজনীয়তা

নিশ্চিত করুন যে আপনার পাইথন ইনস্টল করা আছে (3.8 বা উচ্চতর সুপারিশ করা হয়েছে)। প্রয়োজনীয় প্যাকেজগুলি ইনস্টল করুন:

```bash
# পাইথন স্ক্রিপ্টের জন্য
pip install numpy

# জুপিটার নোটবুকের জন্য (ইমেজ ক্লাসিফায়ার)
pip install jupyter numpy pillow tensorflow
```

অথবা মূল পাঠ্যক্রম থেকে কন্ডা পরিবেশ ব্যবহার করুন:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### উদাহরণসমূহ চালানো

**পাইথন স্ক্রিপ্টের জন্য (.py ফাইল):**
```bash
python 01-hello-ai-world.py
```

**জুপিটার নোটবুকের জন্য (.ipynb ফাইল):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 শেখার পথ

আমরা উদাহরণগুলো নিম্নলিখিত ক্রমে অনুসরণ করার পরামর্শ দিই:

1. **"Hello AI World" দিয়ে শুরু করুন** - প্যাটার্ন চিনাক্তকরণের বেসিক শিখুন
2. **একটি সহজ নিউরাল নেটওয়ার্ক তৈরি করুন** - নিউরাল নেটওয়ার্ক কিভাবে কাজ করে বুঝুন
3. **ছবি শ্রেণিবিভাজনকারী চেষ্টা করুন** - বাস্তব ছবি দিয়ে AI-এর কার্যকারিতা দেখুন
4. **টেক্সট মনোভাব বিশ্লেষণ করুন** - ন্যাচারাল ল্যাঙ্গুয়েজ প্রসেসিং অন্বেষণ করুন
