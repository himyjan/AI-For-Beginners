# ప్రారంభికులకు ఉపయుక్తమైన AI ఉదాహరణలు

స్వాగతం! ఈ డైరెక్టరీలో AI మరియు మెషిన్ లెర్నింగ్‌తో ప్రారంభం కావడానికి మీకు సహాయపడే సరళమైన, స్వతంత్ర ఉదాహరణలు ఉన్నాయి. ప్రతి ఉదాహరణ ప్రారంభికులకు అనువైనదిగా, విపుల వ్యాఖ్యలతో మరియు దశల వారీ వివరణలతో రూపొందించబడింది.

## 🚀 ప్రారంభించడం

### సాధనాలు

Python సంస్థాపించినదని నిర్ధారించుకోండి (3.8 లేదా అంతకప్పుడు సిఫార్సు). అవసరమైన ప్యాకేజీలను ఇన్‌స్టాల్ చేయండి:

```bash
# పైన స్క్రిప్టుల కోసం
pip install numpy

# జూయపిటర్ నోట్బుక్స్ (చిత్ర వర్గీకరించే యంత్రం) కోసం
pip install jupyter numpy pillow tensorflow
```

లేదా ప్రధాన పాఠ్యాంశం నుండి కిందా వాతావరణాన్ని ఉపయోగించండి:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### ఉదాహరణలను నడిపించడం

**Python స్క్రిప్ట్ల కోసం (.py ఫైల్స్):**
```bash
python 01-hello-ai-world.py
```

**Jupyter నోట్‌బుక్స్ కోసం (.ipynb ఫైల్స్):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 అభ్యాస పథం

ఉదాహరణలను క్రమం వారీగా అనుసరించమని మేము సిఫారసు చేస్తున్నాము:

1. **"Hello AI World" తో ప్రారంభించండి** - నమూనా గుర్తింపులో ప్రాథమికాలు తెలుసుకోండి
2. **సరళమైన నర్వ జాలాన్ని నిర్మించండి** - నర్వ జాలాలు ఎలా పనిచేస్తాయో అర్థం చేసుకోండి
3. **Image Classifier ప్రయత్నించండి** - వాస్తవ చిత్రాలతో AI కార్యకలాపాలను చూడు
4. **టెక్స్ట్ భావాన్ని విశ్లేషించండి** - సహజ భాషా ప్రాసెసింగ్ పరిశీలించండి
