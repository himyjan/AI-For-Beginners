## 🚀 सुरु गर्ने तरीका

### आवश्यकताहरू

पक्का गर्नुहोस् कि तपाईंसँग पाइथन इन्स्टल गरिएको छ (3.8 वा माथि सिफारिस गरिएको)। आवश्यक प्याकेजहरू इन्स्टल गर्नुहोस्:

```bash
# पाइथन स्क्रिप्टहरूको लागि
pip install numpy

# ज्यूपिटर नोटबुकहरूको लागि (छवि वर्गीकरणकर्ता)
pip install jupyter numpy pillow tensorflow
```

वा मुख्य पाठ्यक्रमबाट कन्डा वातावरण प्रयोग गर्नुहोस्:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### उदाहरणहरू चलाउने तरिका

**पाइथन स्क्रिप्टहरू (.py फाइलहरू) का लागि:**
```bash
python 01-hello-ai-world.py
```

**जुपिटर नोटबुकहरू (.ipynb फाइलहरू) का लागि:**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 सिकाइ मार्ग

हामी सिफारिस गर्दछौं कि निम्न क्रममा उदाहरणहरू पालना गर्नुहोस्:

1. **"Hello AI World" बाट सुरु गर्नुहोस्** - ढाँचा पहिचानका आधारहरू सिक्नुहोस्
2. **एक सरल न्यूरल नेटवर्क बनाउनुहोस्** - न्यूरल नेटवर्क कसरी काम गर्छ बुझ्नुहोस्
3. **तस्बिर वर्गीकरण प्रयास गर्नुहोस्** - वास्तविक तस्बिरहरूसँग एआईको क्रियाकलाप हेर्नुहोस्
4. **पाठ भावना विश्लेषण गर्नुहोस्** - प्राकृतिक भाषा प्रशोधन अन्वेषण गर्नुहोस्

## 💡 शुरुवातकर्ताका लागि सुझावहरू
