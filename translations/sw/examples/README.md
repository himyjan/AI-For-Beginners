## 🚀 Kuanzisha

### Masharti

Hakikisha umeweka Python (3.8 au juu zaidi inapendekezwa). Toshea vifurushi vinavyohitajika:

```bash
# Kwa maandishi ya Python
pip install numpy

# Kwa daftari za Jupyter (kipanga picha)
pip install jupyter numpy pillow tensorflow
```

Au tumia mazingira ya conda kutoka mtaala mkuu:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### Kuendesha Mifano

**Kwa ajili ya faili za Python (.py):**
```bash
python 01-hello-ai-world.py
```

**Kwa ajili ya daftari za Jupyter (.ipynb):**
```bash
jupyter notebook 03-image-classifier.ipynb
```
