## 🚀 Memulakan

### Prasyarat

Pastikan anda memasang Python (disyorkan 3.8 atau lebih tinggi). Pasang pakej yang diperlukan:

```bash
# Untuk skrip Python
pip install numpy

# Untuk buku nota Jupyter (pengelasan imej)
pip install jupyter numpy pillow tensorflow
```

Atau gunakan persekitaran conda daripada kurikulum utama:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### Menjalankan Contoh

**Untuk skrip Python (.py):**
```bash
python 01-hello-ai-world.py
```

**Untuk nota Jupyter (.ipynb):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 Laluan Pembelajaran

Kami mengesyorkan mengikuti contoh mengikut urutan:

1. **Mulakan dengan "Hello AI World"** - Pelajari asas pengenalan corak
