# ಆರಂಭಿಕರ ಸ್ನೇಹಿ AI ಉದಾಹರಣೆಗಳು

ಸ್ವಾಗತ! ಈ ಡೈರಕ್ಟರಿ ಸರಳ, ಸ್ವತಂತ್ರ ಉದಾಹರಣೆಗಳನ್ನು ಹೊಂದಿದೆ, ಇದು ನಿಮ್ಮನ್ನು AI ಮತ್ತು ಯന്ത്ര ಅಧ್ಯಯನದ ಪ್ರಾರಂಭಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಪ್ರತೀ ಉದಾಹರಣೆ ಆರಂಭಿಕರಿಗಿಷ್ಟು ಸಹಜವಾಗಿದ್ದು, ವಿವರವಾದ ಟಿಪ್ಪಣಿಗಳು ಮತ್ತು ಹಂತ ಹಂತದ ವಿವರಣೆಗಳೊಂದಿಗೆ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

## 🚀 ಪ್ರಾರಂಭಿಸೋಣ

### ಪೂರ್ವಾಪೇಕ್ಷೆಗಳು

Python ಇನ್ಸ್ಟಾಲ್ ಮಾಡಿಕೊಂಡಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ (3.8 ಅಥವಾ ಹೆಚ್ಚು ಶಿಫಾರಸು). ಬೇಕಾಗುವ ಪ್ಯಾಕೇಜುಗಳನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡಿ:

```bash
# ಪೈಥಾನ್ ಲಿಪಿಗಳಿಗೆ
pip install numpy

# ಜುಪಿಟರ್ ನೋಟ್‌ಬುಕ್‌ಗಳಿಗೆ (ಚಿತ್ರ ವರ್ಗೀಕರಣೆ)
pip install jupyter numpy pillow tensorflow
```

ಅಥವಾ ಮುಖ್ಯ ಪಾಠಕ್ರಮದಿಂದ conda ಪರಿಸರವನ್ನು ಉಪಯೋಗಿಸಿ:

```bash
conda env create --name ai4beg --file ../environment.yml
conda activate ai4beg
```

### ಉದಾಹರಣೆಗಳನ್ನು ಚಾಲನೆ ಮಾಡುವುದು

**Python ಸ್ಕ್ರಿಪ್ಟ್‌ಗಳಿಗೆ (.py ಫೈಲ್‌ಗಳು):**
```bash
python 01-hello-ai-world.py
```

**Jupyter ನೋಟ್ಬುಕ್‌ಗಳಿಗೆ (.ipynb ಫೈಲ್‌ಗಳು):**
```bash
jupyter notebook 03-image-classifier.ipynb
```

## 📖 ಕಲಿಕೆಯ ಮಾರ್ಗ

ನಾವು ಕೆಳಗಿನ ಕ್ರಮದಲ್ಲಿ ಉದಾಹರಣೆಗಳನ್ನು ಅನುಸರಿಸುವ ಪರಿಕಲ್ಪನೆಯನ್ನು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ:

1. **"Hello AI World" ನಿಂದ ಪ್ರಾರಂಭಿಸಿ** - ಪ್ಯಾಟರ್ನ್ ಗುರುತಿಕೆಯ ಮೂಲತತ್ವಗಳನ್ನು ಕಲಿಯಿರಿ
2. **ಸರಳ ನೂರಲ್ ನೆಟ್‌ವರ್ಕ್ ನirmaಣ ಮಾಡಿ** - ನೂರಲ್ ನೆಟ್‌ವರ್ಕ್‌ಗಳು ಹೇಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ
3. **ಚಿತ್ರ ವರ್ಗೀಕರಣೆ ಪ್ರಯತ್ನಿಸಿ** - ನಿಜವಾದ ಚಿತ್ರಗಳೊಂದಿಗೆ AI ಅನ್ನು ನೋಡಿ
4. **ಪಠ್ಯ ಭಾವನೆ ವಿಶ್ಲೇಷಿಸಿ** - ನೈಸರ್ಗಿಕ ಭಾಷಾ ಪ್ರಕ್ರಿಯೆಯನ್ನು ಅನ್ವೇಷಿಸಿ

## 💡 ಆರಂಭಿಕರಿಗಾಗಿ ಸಲಹೆಗಳು

- **ಕೋಡ್ ಟಿಪ್ಪಣಿಗಳನ್ನು ಮನಃಪೂರ್ವಕವಾಗಿ ಓದಿ** - ಪ್ರತಿಯೊಂದು ಸಾಲಿನಲ್ಲಿ ಏನು ಮಾಡುವುದೆಂದು ವಿವರಿಸಲಾಗಿದೆ
- **ಪ್ರಯೋಗ ಮಾಡಿ!** - ಮೌಲ್ಯಗಳನ್ನು ಬದಲಿಸಿ ಮತ್ತು ಪರಿಣಾಮವನ್ನು ನೋಡಿ
- **ಎಲ್ಲವನ್ನೂ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಒತ್ತಡಪಡಿಸಿಕೊಳ್ಳಬೇಡಿ** - ಕಲಿಕೆ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ
- **ಪ್ರಶ್ನೆ ಕೇಳಿರಿ** - [ಚರ್ಚಾ ಫಲಕವನ್ನು](https://github.com/microsoft/AI-For-Beginners/discussions) ಬಳಸಿರಿ
